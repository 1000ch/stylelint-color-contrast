# stylelint-color-contrast

:construction:

## Install

```bash
$ npm install --save-dev stylelint-color-contrast
```

## Usage

`.stylelintrc`

```json
{
  "plugins": [
    "stylelint-color-contrast"
  ],
  "rules": {
    "color-contrast/color-contrast": [
      "color",
      "background"
    ]
  }
}
```
